# Research Lab Pattern

**A director coordinating specialized researchers.** For content, analysis, and knowledge work.

```
┌────────────────────────────────────────┐
│         Director (Opus)                │
│    Research planning, synthesis        │
├──────────┬──────────┬────────┬────────┤
│ Analyst  │  Writer  │Scraper │  Fact  │
│ (Sonnet) │ (Sonnet) │(Codex) │ Check  │
│  Data    │  Reports │  Web   │(Codex) │
└──────────┴──────────┴────────┴────────┘
```

## How It Works

1. **Director** defines research questions and methodology
2. **Scraper** collects raw data from web sources
3. **Analyst** processes and interprets data
4. **Fact-Checker** validates claims against sources
5. **Writer** produces final reports
6. Director synthesizes and publishes

## When to Use

- Content marketing at scale
- Competitive intelligence
- Market research automation
- Academic or technical report generation
